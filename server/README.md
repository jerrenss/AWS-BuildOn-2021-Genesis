# Node Server Template

ExpressJS Server Template with basic setup.

# Installation

1. cd into the project

`npm install`

2. Setup your env variables on your `.env` file reference from `.env.example`

# Usage

Run on Development

`npm run dev`

Run on Production
`npm run start`
